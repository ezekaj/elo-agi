# Benchmarks

This document describes ELO-AGI's evaluation methodology and results.

## Overview

ELO-AGI is evaluated through two complementary approaches:

1. **Unit tests** - 3,134 tests across 38 modules verifying correctness of individual components
2. **Internal benchmarks** - Synthetic evaluation suite measuring module-level capabilities

> **Note:** Current benchmarks are internal consistency metrics. They measure whether the architecture's components function correctly according to their design specifications. They are **not** external benchmark results and should not be compared to LLM leaderboard scores.

## Internal Benchmark Suite

Run with: `neuro-bench -o report.html`

The internal benchmark evaluates 6 capability categories using synthetic test scenarios:

| Category | Score | Description |
|----------|-------|-------------|
| Continual Learning | 0.75 | Elastic weight consolidation, replay, and task inference |
| Memory | 0.72 | Working memory, episodic retrieval, and consolidation |
| Abstraction | 0.71 | Symbolic binding, compositional generalization |
| Causal | 0.70 | Intervention calculus, counterfactual reasoning |
| Reasoning | 0.67 | Logical, dimensional, and perceptual reasoning |
| Language | 0.66 | Semantic parsing, discourse modeling |

**Overall Score: 0.697**

### Methodology

Scores are generated by running each module's core algorithms against standardized test inputs and measuring output quality via:
- **Exact match** (50% weight) - correct answers to structured problems
- **Keyword coverage** (30% weight) - relevant concepts mentioned in responses
- **Reasoning chain** (20% weight) - logical structure of explanations

These are deterministic, reproducible scores (seeded with `np.random.seed(42)`).

## Feature Comparison

The website includes a feature-level comparison table against ACT-R, SOAR, and GPT-4. This compares **architectural features** (modularity, local execution, causal reasoning support, licensing), not benchmark scores. The comparison reflects design capabilities, not measured performance.

## Theoretical Foundations

The architecture implements concepts from established cognitive science research:

- **Free Energy Principle** - Friston, K. (2010). "The free-energy principle: a unified brain theory?" *Nature Reviews Neuroscience*, 11(2), 127-138.
- **Global Workspace Theory** - Baars, B. J. (1988). *A Cognitive Theory of Consciousness*. Cambridge University Press.
- **Active Inference** - Friston, K., FitzGerald, T., Rigoli, F., Schwartenbeck, P., & Pezzulo, G. (2017). "Active Inference: A Process Theory." *Neural Computation*, 29(1), 1-49.
- **Dual-Process Theory** - Kahneman, D. (2011). *Thinking, Fast and Slow*. Farrar, Straus and Giroux.
- **Predictive Coding** - Rao, R. P., & Ballard, D. H. (1999). "Predictive coding in the visual cortex." *Nature Neuroscience*, 2(1), 79-87.
- **Elastic Weight Consolidation** - Kirkpatrick, J., et al. (2017). "Overcoming catastrophic forgetting in neural networks." *PNAS*, 114(13), 3521-3526.
- **Structural Causal Models** - Pearl, J. (2009). *Causality: Models, Reasoning, and Inference*. Cambridge University Press.

## Reproducing Results

```bash
# Install
pip install neuro-agi

# Run benchmark suite
neuro-bench -o report.html

# Run all unit tests
pytest neuro-*/tests/ -v

# Run specific module tests
pytest neuro-causal/tests/ -v
pytest neuro-abstract/tests/ -v
```
