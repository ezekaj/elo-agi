# NEURO Brand Guidelines

## Brand Identity

**Name:** NEURO
**Tagline:** Local AI That Learns
**Package:** neuro-agi

## Color Palette

### Primary Colors

| Color | Hex | RGB | Usage |
|-------|-----|-----|-------|
| Deep Space | `#1a1a2e` | rgb(26, 26, 46) | Primary backgrounds, logo bg |
| Electric Cyan | `#00d4ff` | rgb(0, 212, 255) | Accents, highlights, nodes |
| Ocean Blue | `#0099cc` | rgb(0, 153, 204) | Gradient end, secondary accent |

### Neutral Colors

| Color | Hex | Usage |
|-------|-----|-------|
| White | `#ffffff` | Text on dark, primary content |
| Light Gray | `#e0e0e0` | Secondary text |
| Medium Gray | `#888888` | Muted text, taglines |
| Dark Gray | `#666666` | Text on light backgrounds |

### Usage Guidelines

- Use Deep Space (`#1a1a2e`) as primary background
- Electric Cyan (`#00d4ff`) for interactive elements and highlights
- White text on dark backgrounds, Dark Gray on light backgrounds
- Gradient from Electric Cyan to Ocean Blue for neural network nodes

## Typography

### Primary Font
**JetBrains Mono** (monospace)
- Logo text
- Code blocks
- Technical content

Fallbacks: `'SF Mono', 'Menlo', 'Consolas', monospace`

### Secondary Font
**Inter** or system fonts
- Body text
- Documentation
- UI elements

Fallbacks: `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif`

## Logo Usage

### Available Formats

| File | Usage |
|------|-------|
| `logo-icon.svg` | Favicon, app icons, small spaces |
| `logo-full.svg` | Default logo with text |
| `logo-dark.svg` | For dark backgrounds |
| `logo-light.svg` | For light backgrounds |

### Clear Space
Maintain minimum clear space around logo equal to the height of the "N" in NEURO.

### Minimum Size
- Icon: 24x24px minimum
- Full logo: 120px width minimum

### Don'ts
- Don't stretch or distort the logo
- Don't change the colors outside brand palette
- Don't add effects (shadows, glows) to the logo
- Don't place on busy backgrounds without contrast

## Visual Elements

### Neural Network Motif
The logo features a simplified neural network representing:
- 3 input nodes (data, context, learning)
- 2 hidden nodes (processing, reasoning)
- 1 output node (intelligent response)

This represents the NEURO architecture: Perceive → Think → Act

### Icon Style
- Rounded, friendly appearance
- Consistent stroke width (1.2-1.5px)
- Gradient fills for depth
- Subtle connection lines at 50% opacity

## Voice & Tone

### Brand Personality
- **Technical** but accessible
- **Intelligent** but humble
- **Local-first** and privacy-focused
- **Learning** and adaptive

### Writing Style
- Clear and concise
- Avoid jargon unless necessary
- Focus on benefits, not just features
- Emphasize privacy and local operation

### Example Phrases
- "Your AI, on your machine"
- "Gets smarter the more you use it"
- "No cloud, no data sharing, no API costs"
- "26 cognitive modules inspired by neuroscience"
