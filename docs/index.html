What It Does

ELO-AGI (NEURO) is a comprehensive neuroscience-inspired artificial general intelligence system that implements 38 cognitive modules modeled after human brain architecture. It spans four architectural tiers: Cognitive, Infrastructure, Support, and AGI.

The system runs entirely locally using Ollama as its LLM backend, ensuring complete privacy with zero cloud dependency and zero API costs. It learns from your codebase, adapts to your patterns, and provides intelligent coding assistance without ever transmitting your data.

Drawing from cognitive science research, NEURO implements global workspace theory, predictive coding, dual-process reasoning, continual learning with catastrophic forgetting prevention, causal reasoning, and meta-cognition -- creating a unified cognitive architecture that mirrors the modularity and integration of the human brain.
